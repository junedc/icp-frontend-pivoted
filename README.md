# ICP Frontend (Vue 3 + Pinia + Vue Query + Tailwind)

## Setup
```bash
npm install
cp .env.example .env
npm run dev
```

## Running (Important)
#### The backend is running in Sail to endpoint will be laravel.test.  Please change accordingly based on your backend setup
```bash
VITE_API_BASE_URL=http://laravel.test   
```
