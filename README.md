# Vue Shop App (Vue 3 + Pinia + Vue Query + Tailwind)

This app matches your `/api/login` response shape:

```json
{
  "status": "Request was successful",
  "data": { "user": { "id": 1, "name": "June", "email": "..." }, "token": "11|..." },
  "message": null
}
```

## Setup
```bash
npm install
cp .env.example .env
npm run dev
```
